# Pierre Policy & Strategy Consulting

Source for the [Pierre Policy & Strategy Consulting](https://github.com/Dakpi11/pierrepolicy-website) website — a static site presenting independent research and policy analysis on conflict, security, and international affairs.

## Structure

- `index.html` — homepage
- `papers.html` — list of published papers and briefs
- `about.html` — firm and founder background
- `style.css` — shared styles for all pages

## Running locally

This is a static HTML/CSS site with no build step. Open `index.html` directly in a browser, or serve the folder locally, e.g.:

```
npx serve .
```

## Deployment

Push changes to `main`; the site is served directly from the repository files.
